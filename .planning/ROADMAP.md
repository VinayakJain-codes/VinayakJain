# Project Roadmap

The roadmap divides the project into sequential phases.
Each phase has a clear goal and maps to specific requirements.

## Current Milestone (v1.0 - Initial Launch)

### Phase 1: Foundation & Hero Section
- **Goal**: Set up the Next.js project with Tailwind CSS, Framer Motion, layout and fonts, and build the Hero section.
- **Requirements**: REQ-1, REQ-2

### Phase 2: About & Works Sections
- **Goal**: Build the About section with tech chips, and the primary Works section (4 project cards).
- **Requirements**: REQ-3, REQ-4

### Phase 3: Experience, Skills & Vicinix Sections
- **Goal**: Implement the Timeline, Skills tag cloud, and Freelance (Vicinix) sections.
- **Requirements**: REQ-5, REQ-6

### Phase 4: Contact, Polish & Mobile Responsiveness
- **Goal**: Add the Contact section, ensure mobile responsiveness across the entire site, and apply metadata/SEO optimizations.
- **Requirements**: REQ-7

### Phase 5: Scroll Animations & Email Template
- **Goal**: Add cinematic scroll-driven parallax, staggered card reveals, site-wide section fade-ups, animated text reveals, and a mailto template pre-filled with a structured hiring inquiry.
- **Requirements**: REQ-8, REQ-9

## Future Milestones
- Phase 2 additions (Notes, Testimonials, Dark/Light toggle)
